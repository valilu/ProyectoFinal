# Proyecto Final - Idea Inicial

**Q1. ¿En qué organismo o grupo de organismos vas a trabajar?**  
Voy a trabajar con tortugas marinas, específicamente con la especie *Chelonia mydas* (tortuga verde), y otras especies cercanas como *Eretmochelys imbricata* (tortuga carey), *Lepidochelys olivacea* (tortuga golfina) y *Dermochelys coriacea* (tortuga laúd).

**Q2. Brevemente describe qué piensas hacer en tu proyecto**  
Mi objetivo es construir una filogenia que muestre las relaciones evolutivas entre diferentes especies de tortugas marinas, usando secuencias obtenidas de bases de datos públicas. Esto permitirá visualizar cómo se relacionan entre sí, identificar posibles eventos de divergencia evolutiva y comparar la diversidad genética entre especies.

**Q3. ¿Qué programas voy a usar en mi proyecto?**  
Voy a usar:
- NCBI para obtener las secuencias mitocondriales
- MUSCLE para alineamientos múltiples
- IQ-TREE para inferir el árbol filogenético
- FigTree para visualizar el árbol
